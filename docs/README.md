# Documentacao PoliPadaria

Esta pasta concentra a documentacao textual principal do projeto PoliPadaria, organizada para uso academico, consulta tecnica e apoio a manutencao do backend.

## Estrutura

- [descricao_projeto.md](/home/enoch/projects/PoliPadaria/docs/descricao_projeto.md): visao geral, objetivos, escopo e decisoes de projeto.
- [modelo_dados.md](/home/enoch/projects/PoliPadaria/docs/modelo_dados.md): descricao das tabelas, chaves, restricoes e relacionamentos.
- [consultas.md](/home/enoch/projects/PoliPadaria/docs/consultas.md): consultas SQL e chamadas REST realmente coerentes com a implementacao atual.
- [geracao_mock.md](/home/enoch/projects/PoliPadaria/docs/geracao_mock.md): estrategia de geracao automatizada de dados mock e criterios de realismo.
- [relatorio_atividades.md](/home/enoch/projects/PoliPadaria/docs/relatorio_atividades.md): relatorio tecnico das atividades de desenvolvimento.
- [manual_usuario.md](/home/enoch/projects/PoliPadaria/docs/manual_usuario.md): guia pratico para executar, preparar e testar o sistema.

## Observacao

A documentacao foi produzida a partir da leitura do codigo existente no backend, do schema SQLite, dos scripts de seed e do gerador de mock data. Quando havia diferenca entre a ideia geral do projeto e o que esta implementado no repositorio, foi priorizada a implementacao real.
